Includes 5 detailed examples for nested lists and 5 detailed examples for tables with step-by-step guidance.

HTML Lists Recap

  • ol - Ordered list (numbered).
  • ul - Unordered list (bulleted).

Nested lists: Lists within lists to represent hierarchical data.

Introduction to HTML Tables

HTML tables are used to present tabular data with rows and columns:

  • table tag defines the table
  • tr tag defines a table row
  • th tag defines a table header cell (bold and semantic)
  • td defines a table data cell
  • thead, tbody, and tfoot define semantic sections of the table

Example: Simple table structure

Item Quantity Price
Apple 10 $1
Orange 5 $0.5
Total 15 $1.5

Tips

  • Use semantic tags like thead, tbody, and tfoot for clarity & accessibility.
  • Use th for headers to aid screen readers and SEO.
  • Indent and comment code clearly for readability.
  • Use borders temporarily (e.g., border="1") for visible debugging of table layout.
