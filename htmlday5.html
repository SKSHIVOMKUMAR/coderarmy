1. Embedding Audio Files

The audio tag adds sound/music to your webpage.
Controls: Add controls for play/pause buttons.
Tip: Use source for multiple audio formats to ensure wider browser support.


2. Embedding Video Files

The video tag shows videos directly in HTML.
Controls: Add controls for video buttons.
Poster: Show a placeholder image.
Tip: Use source for multiple formats.

To autoplay/mute:
<video controls autoplay muted> ... </video>

3. Embedding YouTube & External Videos

Use iframe tag for videos from YouTube or other sites.
Allow Fullscreen: Add allowfullscreen attribute.
Example:

  • Replace VIDEO_ID with the correct video ID from the URL.
  • Always check you are using the /embed format for YouTube.

4. Folder Structure for Multi-Page Website

  1. Organize all HTML files (e.g. index.html, about.html, contact.html) in one project folder.
  2. Store media files (audio, video, images) in separate folders for easy access.
  3. Correctly link files using relative paths.

5. Linking Multiple Pages

Use the a(anchor tag )tag to link pages. Example navigation:

  • Place the links in the header of every page for easy navigation.
  • Use relative paths (folder-name/file-name.html) for links within folders.

How to Deploy Website using Netlify?

  1. Create project folder and keep all HTML files inside it.
  2. Sign up for a free account on netlify.com
  3. Click "Add new site" "Deploy manually".
  4. Drag and drop your entire project folder onto Netlify deploy area.
  5. Wait for upload; you get a LIVE project URL to share.
  6. To update, just re-upload new files to the same project on Netlify.
  7. Your site is automatically secured with HTTPS.
